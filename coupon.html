Реферальная программа компании Acryl

Стать участником программы может любой пользователь сети Интернет, у которого имеется зарегистрированный аккаунт на платформе Acryl Client (кошелек) и алиас* для этого кошелька.
*алиас - это дополнительное “простое” имя для уже существующего адреса кошелька.
Основные положения реферальной программы Acryl
Каждый пользователь, у которого есть аккаунт на Acryl Client (кошелек), может сгенерировать личную реферальную ссылку, чтобы поделиться с друзьями, коллегами, знакомыми и незнакомыми пользователями сети Интернет, которые заинтересованы в покупке Acryl Node. Такой пользователь будет считаться рефералом. Официальный представитель платформы, заключивший договор с платформой Acryl будет считаться дистрибьютором.
Вознаграждение по реферальной программе начисляется в Acryl Coupon в следующих случаях:
  • В случае привлечения клиента, с которым дальше работает отдел продаж, реферал считается ледогенератором, и получает 2 Acryl Coupon за каждое проданное устройство платформой Acryl, включая повторные покупки клиента.
  • В остальных случаях вознаграждение рефералов и дилеров рассчитывается менеджером в зависимости от степени участия в процессе привлечения покупателя, но не может превышать 10 Acryl Coupon.
Контакт для связи с менеджером реферальной программы: sales@acrylminer.com
Контакт для связи с менеджером дилерской программы: distr@acrylminer.com
Начисление вознаграждения происходит 10, 20, 30 числа каждого месяца на кошельки, у которых есть алиасы.
Проверить текущее количество уже зачисленных купонов Acryl Coupon участник может в приложении Acryl Client во вкладке Кошелек, где расположены все ассеты.
Администрация компании оставляет за собой право менять условия реферальной программы с условием преждевременного оповещения пользователей.
